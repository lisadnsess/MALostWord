## 功能列表
- 永远战线（暂时不支持换位）

## 使用说明
- 战斗文件在 ./agent/FightStrategy/ 中
- 与高级设置里头填写战斗方式以及战斗重复次数
- 格式参考fight1.json,为完整的一轮

## 后续功能
- 自动完成每日任务（接口还没弄）
- 启动/关闭游戏（接口还没弄）
